# Spritesheet Quality Checklist

- [ ] Frames bem alinhados no grid
- [ ] Sem pixels soltos ou ruído visual
- [ ] Transparência correta (sem halos brancos)
- [ ] Tamanho otimizado para WebGL (potências de 2 preferencialmente)
- [ ] Nome do arquivo segue convenção (Tipo_Descricao-Sheet.png)
- [ ] Frame 0 é o idle/default
- [ ] Animação death tem fade out ou transição
- [ ] Spritesheet inclui todas as direções necessárias
- [ ] Sem espaços vazios entre frames no sheet
- [ ] Cor do decalque de morte definida por tipo de entidade
