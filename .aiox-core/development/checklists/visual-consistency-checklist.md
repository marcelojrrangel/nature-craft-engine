# Visual Consistency Checklist

- [ ] Todos os sprites usam resolução consistente (16x16 grid)
- [ ] Paleta de cores harmônica entre ambiente, NPCs e jogador
- [ ] Spritesheets têm padding e framing consistentes
- [ ] Animações têm frame rate consistente (6-12 fps para pixel art)
- [ ] Hitbox visual corresponde à hitbox física
- [ ] Efeitos de morte (decalque) seguem padrão de cor por tipo
- [ ] Ícones de itens seguem estilo pixel art unificado
- [ ] Partículas (hit, craft, collect) usam paleta do jogo
- [ ] Iluminação (Light2D) respeita a atmosfera do jogo
- [ ] Safe Zone visível com estilo consistente
