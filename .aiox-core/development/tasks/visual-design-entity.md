# Visual Design: New Entity

## Objetivo
Planejar e projetar o visual de uma nova entidade (NPC, recurso, item, ou objeto).

## Passos
1. **Definir propósito** — Qual o papel da entidade no jogo?
2. **Referências** — Coletar referências visuais do estilo Pixel Crawler
3. **Tamanho** — 32x62 (árvores), 64x64 (jogador), 16x16 (ícones), etc.
4. **Paleta** — Definir paleta de 8-16 cores
5. **Spritesheet** — Planejar grid de frames (colunas x linhas)
6. **Estados** — idle, walk, attack, hurt, death
7. **Hitbox** — Definir bounds físicos
8. **Decalque de morte** — Cor e forma da mancha no chão
