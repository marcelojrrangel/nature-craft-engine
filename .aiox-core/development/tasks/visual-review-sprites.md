# Visual Review: Sprites

## Objetivo
Auditar todos os spritesheets do jogo quanto a consistência de estilo, resolução, paleta e qualidade pixel art.

## O que verificar
1. **Estilo consistente** — Todos os sprites seguem o mesmo padrão pixel art?
2. **Densidade de pixels** — Mesma resolução e escala entre entidades similares?
3. **Paleta** — Cores harmônicas, sem banding, número limitado de cores
4. **Framing** — O sprite está centralizado? Padding consistente?
5. **Hitbox alignment** — O visual corresponde à hitbox real?
6. **Formato** — PNG otimizado? Transparência correta?

## Assets a revisar
- `public/assets/env/` — Todos os assets de ambiente
- `public/assets/player/` — Sprites do jogador
- `public/assets/env/enemies/` — Sprites de inimigos
- `public/assets/env/chicken/`, `crab/`, `rabbit/` — Sprites de NPCs
