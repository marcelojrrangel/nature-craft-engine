# 🎮 Game Designer Skill

Use this skill when reviewing or designing game visuals for a Phaser pixel art project.

## When to use
- Reviewing spritesheet quality and style consistency
- Designing new entity visuals (NPCs, resources, items)
- Auditing animation states and transitions
- Planning visual effects (particles, lighting)
- Creating pixel art style guidelines
